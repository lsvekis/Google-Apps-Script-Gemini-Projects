# AI Google Docs Content Refiner (Apps Script + Gemini)

Refine, rewrite, summarize, and improve Google Docs content using Gemini + Apps Script.

## Setup
1. Open a Google Doc
2. Extensions → Apps Script
3. Replace project files with this repo
4. Add Gemini API key in GeminiHelpers.gs
5. Run testGeminiConnection()
6. Reload document

Menu:
AI Tools → AI Content Refiner
